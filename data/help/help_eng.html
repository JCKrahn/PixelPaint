PixelPaint: Help

Create a new Image

Go to File/New. Customize your new Image by setting its width/ height and background (you can get additional info by clicking the '?' - then click on something that you want to know more about).

Open an Image

Go to File/Open - PixelPaint supports JPG (with 3 channels) and PNG (with 4 channels).

Save your Image

Go to File/Save or Save as. If you want your Image to have transparency - save it as a PNG.

Paint-Window

(Pops up if you have created/ opened an Image)

Controls:
Left-click the Image to work with the tool you selected, if no tool is selected you can drag the Image around.
Right-click the Image to drag it around (also works if you have a tool selected).
Hover over the Image and press Ctrl + mouse-wheel-up/ mouse-wheel-down to zoom in/ out.
Press Z/ Y to undo/ redo changes.
Press S to quick-save the Image.

Tool-Bar

Select the tool that you want to work with by left-clicking it.

Color-Palette

Select the color that you want to work with by left-clicking it.
Right-click a color to change it.
(Color1 will also be referred to as Main-Color)

Fill-Tool

The Fill-Tool lets you fill an area with your selected color.
To cancel the Fill-Operation click the close button of the Paint-Window (the window won't close in this specific case; if you want to close the window click the close button again).
Settings (Settings/PaintWindow Settings):
You can enable/ adjust the Fill-Tolerance which determines how much the pixel colors can differ to be still filled.
You can disable the Fill-Algorithm Visualization to gain performance.

Image-Maxsize

The Image Maxsize prevents you from experiencing performance issues and crashes (PixelPaint is simply not designed to handle big images). You can disable it (Settings/PaintWindow Settings) but be warned.
