## Build:
### requires:
- python 2.7
- packages:
  - pywin32(= 227)
  - numpy(= 1.16.6)
  - opencv-python(= 4.1.2.30)
  - pygame(= 1.9.6)
  - pathlib2(= 2.3.5)
  - cx-Freeze(= 5.1.1)
  - python-qt5(= 0.1.10)
  
### how to build:
- run exe_setup.py 
- modify file-paths in PixelPaint_installer_setup.iss
- run PixelPaint_installer_setup.iss with [InnoSetup](https://www.jrsoftware.org/isinfo.php)
